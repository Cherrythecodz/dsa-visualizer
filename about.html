About This Project

The DSA Visualizer is an interactive web application designed to help students and enthusiasts understand Data Structures and Algorithms through visualizations. Instead of just reading code, you can see algorithms in action — step by step!

✨ Features

  • Sorting Algorithms – Bubble, Insertion, Selection, Merge, Quick, Heap
  • Searching Algorithms – Linear Search, Binary Search
  • Binary Search Trees – Insertion, Deletion, Traversals (Inorder, Preorder, Postorder)
  • Graph Algorithms – BFS, DFS, Dijkstra (with canvas/grid visualization)

📖 Purpose

This project was built as part of my 3rd Semester coursework. The goal is to make learning DSA more engaging by providing interactive animations that help students understand how algorithms work behind the scenes.
